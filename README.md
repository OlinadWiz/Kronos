# K.R.O.N.O.S. - Kanal Routing Optimized Network On Stremio

Addon configuratore di canali TV per Stremio che supporta:

- 📺 Liste M3U multiple
- 📋 Guida EPG (Electronic Program Guide)
- 🔄 Proxy resolver per stream
- 🎯 Filtri per gruppi di canali

## Utilizzo

1. Accedi all'interfaccia web dello Space
2. Configura le tue liste M3U
3. Aggiungi EPG (opzionale)
4. Seleziona i gruppi di canali
5. Genera il link per Stremio
6. Installa l'addon in Stremio

## Configurazione

L'addon è pronto all'uso. Tutte le configurazioni vengono gestite tramite l'interfaccia web.

## Note

- Versione: 1.5.7
- Porta locale: 7000 
- Porta: 7860
- Node.js: >= 18
